# Dictation Practice App

English listening practice application with dictation exercises, levels, and gamification.

## Project Type
Web application - MVP planning phase

## Core Features
- Dictation practice by CEFR levels (A1-C2)
- Post-exercise evaluation scoring
- Streak system (GitHub-style contribution graph)
- Progress tracking dashboard
- Audio content with transcripts

## Development Guidelines
- Mobile-first responsive design
- Accessibility compliance (WCAG 2.1)
- Offline-capable PWA
- Performance-optimized audio streaming

## Directory Structure
```
/docs              - Documentation & wireframes
/src               - Source code (TBD)
/public            - Static assets
/content           - Audio content & transcripts
```
